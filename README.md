# Syntecxhub Internship Project

This repository contains the `Sample.ipynb` notebook built during my Syntecxhub internship. It demonstrates key data manipulation, analysis, and workflow practices using pandas and Jupyter.

## Project structure
- `Sample.ipynb` - main notebook
- `data/` - example datasets (`bios.csv`, `results.csv`, `results.feather`, etc.)
- `warmup-data/coffee.csv` - sample dataset for learning pandas

## What I learned (major achievements)
- Git & GitHub version control (init, add, commit, push, remote setup)
- Loading data from CSV, Parquet, Excel
- Dataframe inspections: `head()`, `tail()`, `info()`, `describe()`, `shape`
- Row/column selection with `.loc`, `.iloc`, `.at`, `.iat`
- Filtering with conditions, boolean masks, regex string queries
- Column creation and update, dropping columns, row-level edits
- Sorting, grouping/aggregation, `query()` API
- Preparing reproducible analysis in notebooks
- Saving results (`.csv`, `.feather`) and sharing insights
